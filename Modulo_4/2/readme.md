# Reconocedor de emociones

Utilizando tensorflow, entrenamos un modelo de machine learning para reconocer 6 distintos tipos de emociones

<ul>
  <li>Enojado</li>
  <li>Disgustado</li>
  <li>Triste</li>
  <li>Feliz</li>
  <li>Natural o Serio</li>
  <li>Sorprendido</li>
</ul>
